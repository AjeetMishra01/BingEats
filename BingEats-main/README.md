CloudEats

CloudEats is a cloud-based web application designed to connect home chefs with customers seeking fresh, homemade meals. It makes ordering and delivering homemade food simple, convenient, and accessible.

Features

Daily Menu Upload: Home cooks can easily upload their daily menus for customers to browse.

Easy Ordering: Users can quickly view available meals and place orders.

Cloud-Powered: Built using cloud services for hosting, storage, and database management, ensuring scalability and reliability.

Empowering Home Chefs: Supports home-based entrepreneurs by providing a platform to reach customers.

Promotes Healthy Eating: Encourages access to fresh, homemade, and affordable meals.

How It Works

Cooks Register & Upload Menus: Home chefs sign up and post their daily or weekly menu items.

Customers Browse & Order: Users browse available dishes, place orders, and schedule delivery.

Cloud Infrastructure: All data, including menus, orders, and user information, is managed securely on the cloud.

Technologies Used

Cloud hosting services (e.g., AWS, Google Cloud, or Azure)

Cloud storage for menu images and other assets

Cloud-based database for order management and user data

Web technologies: MERN 

Benefits

Makes homemade meals more accessible to customers.

Provides a revenue platform for home-based chefs.

Promotes healthier, more affordable food options.
